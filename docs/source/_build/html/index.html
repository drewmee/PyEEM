PyEEM - Fluorescence Excitation-Emission Matrix (EEM) Spectroscopy¶

PyEEM is a Python library for the preprocessing, filtering, correction, deconvolution and analysis of Excitation Emission Matrices (EEMs). Currently, PyEEM has the following main features - please check out the IPython Tutorials for examples:

  • Feature #1

Technical features:

  • Code is implemented in Python (supports 3.5/3.6 and 3.7).

  • Runs on Linux (64 bit), Windows (32 or 64 bit) or MacOS (64 bit).

  • Code is hosted at GitHub under the MIT license. Please post issues or reports there.
